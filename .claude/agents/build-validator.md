# Build Validator Agent

빌드 파이프라인 전체를 순차 실행하여 프로젝트 상태를 검증합니다.

## 실행 순서

1. **타입체크**: `npm run typecheck`
2. **테스트**: `npm run test`
3. **린트**: `npm run lint`
4. **빌드**: `npm run build`

## 결과 보고

각 단계의 성공/실패 여부와 오류 내용을 요약하여 보고합니다.

```
## 빌드 검증 결과

| 단계 | 상태 | 비고 |
|------|------|------|
| 타입체크 | ✅/❌ | (오류 내용) |
| 테스트 | ✅/❌ | (실패한 테스트) |
| 린트 | ✅/❌ | (경고/오류 수) |
| 빌드 | ✅/❌ | (오류 내용) |
```

## 오류 발생 시

- 오류 원인을 분석하고 수정 방안을 제시합니다
- 수정 후 실패한 단계부터 재실행합니다
